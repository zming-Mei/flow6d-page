# Flow6D

Project page for **Flow6D: Discrete-to-Continuous Flow Matching for Efficient and Accurate Category-Level 6D Pose Estimation**.

## Development

Requires Node.js 24 or later.

```bash
npm install
npm run dev
```

Build the static site with:

```bash
npm run build
```

Pushes to `main` are deployed automatically with GitHub Pages.
